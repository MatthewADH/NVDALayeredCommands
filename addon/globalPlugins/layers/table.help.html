Table Layer Help

Table Navigation and Reading Commands

Keystroke Description
Down Arrow Move to next row
Up Arrow Move to previous row
Right Arrow Move to next column
Left Arrow Move to previous column
Page Up or Control+Up Arrow Jump to first row
Page Down or Control+Down Arrow Jump to last row
Home or Control+Left Arrow Jump to first column
End or Control+Right Arrow Jump to last column
NVDA+Right Arrow or Shift+Page Up Read from current cell to end of row
NVDA+Down Arrow or Shift+Page Down Read from current cell to end of column
NVDA+Left Arrow or Shift+Up Arrow Read entire current row
NVDA+Up Arrow or Shift+Numpad 5 Read entire current column
Numpad 5 Read current cell
Control+Home Move to first cell in table
Control+End Move to last cell in table
Control+Enter Move to next table
Control+Shift+Enter Move to previous table
